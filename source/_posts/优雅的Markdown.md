## 反斜杠的使用

使用反斜杠可以避免文本中的符号被当作markdown标识符而发生不必要的转换

例如
```
\!
\()
\**不是粗体**
\# 不是一级标题
```
\!
\()
\**不是粗体**
\# 不是一级标题

## 使用缩进
```
不断行的空白格  &nbsp;  或  &#160;
半角的空格      &ensp;  或  &#8194;
全角的空格      &emsp;  或  &#8195;

示例:
&ensp; &ensp今天天气好啊
&emsp;&emsp;感觉倍爽啊
```
&ensp;&ensp;今天天气好啊
&emsp;&emsp;感觉倍爽啊

<u>***不要忘了英文的分号***</u>
