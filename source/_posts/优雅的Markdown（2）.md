# Markdown编辑器使用-MarkdownPad2

## 快捷键

|     |     |
|:---:|:---:|
|粗体|ctrl+B|
|斜体|ctrl+I|
|引用|ctrl+Q|
|代码|ctrl+K|
|超链接|ctrl+L|
|图片|ctrl+G|
|无序列表|ctrl+U|
|有序列表|ctrl+shift+O|
|水平标尺|ctrl+R|
|时间戳|ctrl+T|
|撤销|ctrl+Z|
|重做|ctrl+Y|
